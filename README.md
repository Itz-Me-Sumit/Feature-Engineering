# 🛠️ Feature Engineering

This repository covers my **Feature Engineering** practice — transforming raw data into meaningful features that improve model performance, with a focus on understanding *why* each technique works and *when* to apply it.

Part of my larger [Data Science & ML Journey](https://github.com/Itz-Me-Sumit/Data-Science).

---

# 🧠 What's Inside

### 🔧 Feature Extraction
* Extracting meaningful features from raw/structured data
* Domain-driven feature creation (e.g., Haversine distance, cyclical encodings)

### 🧹 Data Cleaning & Transformation
* Numeric Transformers
* Categorical Transformers
* Handling Mixed Feature Types
* Composite Transformers & Chaining Transformers (Pipelines)

### 🎯 Feature Selection
* Filter-based Feature Selection
* Wrapper-based Feature Selection
* Sequential Feature Selection

### 📉 Dimensionality Reduction
* PCA (Principal Component Analysis)
* Handling Class Imbalance
* Feature Distribution Visualization

---

# ⚙️ Approach

✔️ Understanding the purpose behind each transformation, not just applying it
✔️ Building reusable pipelines with Scikit-Learn
✔️ Practicing on real, messy datasets
✔️ Connecting feature engineering decisions to downstream model performance

---

# 🛠️ Tech Stack
* Python
* Pandas
* NumPy
* Scikit-Learn (Pipelines, Transformers, PCA)
